# 完整生产级技术栈
## FastAPI 后端
- 异步接口，附带完整 Swagger 接口文档
- 基于 Pydantic 模型实现请求与响应数据校验
- 依赖注入方式管理数据库会话
- 生产级可靠的异常处理中间件
- 用于运维监控的健康检查接口

## PostgreSQL 数据库
- 适配学术论文元数据的优化数据表结构
- 借助JSONB字段实现灵活化文档存储
- 合理建立索引，提升查询速度
- 连接池机制适配多用户并发访问
- 支持数据表结构迭代升级的数据库迁移方案

## OpenSearch 检索集群
- 适配专业学术术语的自定义分词解析器
- 融合BM25关键词检索与向量相似度检索的混合搜索
- 专为文档检索优化的索引模板
- 集群运行状态监控与告警机制
- 检索查询性能调优

## Apache Airflow 工作流调度
- 依托有向无环图实现论文数据自动化采集
- 任务重试机制与异常故障处理
- 任务依赖关系统筹管理
- 可视化流水线运行状态监控面板
- 可弹性扩容的任务执行能力

## Ollama 大模型容器
- 本地部署大模型推理，无需依赖外部接口
- 硬件资源分配与运行性能调优
- 面向问答场景的开放调用接口
- 坚守数据隐私的本地化AI运算

## Docker Compose 服务编排
- 统一管理各项服务依赖关系
- 统一配置服务健康检测规则
- 网络隔离部署，提升整体安全性
- 数据卷挂载实现数据持久化存储
- 集中化环境变量统一管理
